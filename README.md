# IRAF: The Iterative Refinement Agentic Framework. 

A Git-native, 0-cost architecture for self-improving AI ecosystems. Built for transparency, auditable through every commit.

**Live self-building landing page for the Infinite Recursion Agentic Framework.**

This repository demonstrates IRAF in action: a minimal set of Markdown blueprints + clever prompting that turns any Git repo into a self-sustaining, self-improving AI agent system.

### How It Works
1. Bootstrap prompt activates the agents.
2. Agents read `agents.md`, `skills.md`, `clauses.md`.
3. They build/improve the landing page (`index.html`).
4. Evaluator scores the result.
5. If score < 90, they refine the blueprints **and** the page → recursion loop.
